A professional-grade real-time stock market visualization and analysis dashboard built with Python.

## ✨ Features
* **Real-Time Data:** Fetches live market data via `yfinance`.
* **Dynamic Visualization:** Interactive candlestick charts using `Plotly`.
* **Custom Timeframes:** User-controlled lookback periods (1-365 days).
* **Automated Metrics:** Instant calculation of price change and period highs.
